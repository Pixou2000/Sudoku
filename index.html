Sudoku — Aide

        
      


      

        
Version : 0.1.0


        
Principe

        

          Le but est de compléter la grille de 9×9 avec les chiffres de 1 à 9,
          sans répétition dans une ligne, une colonne ou un bloc 3×3.
        


        
Démarrage

        
    
          
  • Grille vide : crée une grille vide à remplir manuellement.
    • 
          
  • Générer : crée une nouvelle grille automatique selon le niveau choisi.
    • 
          
  • Valider grille : passe du mode préparation au mode jeu.
    • 
        


        
Mode jeu

        
    
          
  • Candidats : OFF / ON : active ou désactive la saisie des candidats.
    • 
          
  • Cands sélection : crée les candidats pour les cellules sélectionnées.
    • 
          
  • Tous cands : crée les candidats pour toute la grille.
    • 
          
  • Stats grille : analyse la grille et affiche son niveau et ses statistiques sans modifier la partie.
    • 
          
  • Undo : annule la dernière action.
    • 
          
  • Effacer : efface les chiffres des cellules sélectionnées.
    • 
          
  • Effacer Candidats : efface seulement les candidats.
    • 
        


        
Sélection

        
    
          
  • Clic : sélection d’une cellule.
    • 
          
  • Glisser : sélection rectangulaire de plusieurs cellules.
    • 
          
  • Double clic / double tap : sélection des mêmes valeurs et candidats associés.
    • 
        


        
Sauvegarde

        
    
          
  • Save : exporte la partie en fichier JSON.
    • 
          
  • Load : recharge une partie sauvegardée.
    •
